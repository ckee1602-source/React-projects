# Chatbot Project
ðŸ”´This project is a React-based chatbot application that implements a real-time messaging interface using functional components and React Hooks (useState, useEffect, useRef). It dynamically renders user and bot messages with conditional styling, profile avatars, and automatic scroll-to-bottom behavior for seamless conversation flow. Chatbot responses are generated via an external chatbot utility, demonstrating efficient state management and component-based frontend architecture without a backend dependency.

## Features 
* Real-time chat interface enabling users to send and receive messages instantly.
* Dynamic message rendering with conditional alignment and profile avatars for user and chatbot messages.
* Automatic scroll-to-bottom functionality to display the latest messages.
* Controlled input field with state management for handling user text input.
* Integration of an external chatbot utility to generate automated responses.
* Component-based architecture for better code reusability and maintainability.
* Responsive and clean user interface styled using CSS.
